<**_Python编程:从入门到实践_**>
的一个终期学习实践项目

<hr>

具体的原理这里不过多做介绍

项目完全由python语言编写,依赖于***pygame***这个外部库,所以使用者请先安装好相应环境

<hr>

### 运行说明

本项目运行只需将本库克隆后运行alien_invasion.py文件即可

<hr>

### 操作说明


1. 方向键的左右控制飞船左右移动
2. 按键空格(space)即可发射子弹
3. 按键q退出游戏
